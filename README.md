# DrChipmunk's EggHub

Hi! I'm DrChipmunk. This is a site to host all of my custom Magic: the Gathering designs.

(If you're looking for instructions on how to build an EggHub like this one, go to https://github.com/magictheegg/mse-hub/)

## My Sets

### Perennial Ultimatum (PRU)

This is my first set, intended mainly for draft and made in the late draft booster style (which is an era of design I'm particularly fond of). It is based on a TTRPG campaign I once ran about the magical land of Faerie, the creatures that lived there, the empire of humans that found them, and how they fought back. As my first work, it's somewhat rough around the edges but I stand by the draft structure here. 

### Errors in the Weft (ERR)

My proudest work; this is a battlecruiser Magic set about the fall of a digital utopia. This set was accepted into Revolution Custom Standard. The story is based on works like Transistor, Twilight Mirage, and 17776, search "tag:story" to find the short stories I have written. This set is polished, unique, and expresses a lot of what I love about Magic, although I certainly had a lot to learn about designing for constructed formats.

### Escape From Limbo (LIM)

A set made in 17 days for the 2025 Summer Santa MSE event. Every card in the set has the word "sacrifice" or "graveyard" on it. A lot of rough work here, but I think it looks surprisingly normal for having such a strange restriction. You may be familiar with the Escape From Limbo search engine for Revolution Custom Standard. That was originally an EggHub made for this set so I could show it anonymously during the Summer Santa event before I repurposed it. 

https://escapefromlimbo.github.io/

### Blood and Velvet (BLV) 

Another set based on a TTRPG campaign; this one a classy, corrupt, and discreet affair about rules, secrets, and an endless city. Macabre in tone, the gameplay is based around a unified theme of "bluffing". Intended for Revolution Custom Standard and still in development.

https://drchipmunk.itch.io/blood-and-velvet

### Clearingsong (CLE)

Inspired by cubes and core sets, this is a wholesome set with a wide variety of overlapping themes and synergies. The theme is "diversity"; the story is about young hatchlings banding together to mend rifts between their hidebound elders. Intended for Revolution Custom Standard and still in development.
